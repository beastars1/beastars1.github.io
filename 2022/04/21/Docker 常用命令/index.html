docker run的额外参数

  • –name 给当前运行的容器起一个名字

    docker run --name mytomcat -d tomcat

  • -d 后台运行容器

  • -p 用于端口映射

    -p src:dest

    src 为物理主机端口，dest 为容器的端口

    -p 8888:9999 就是将物理主机的 8888 端口映射到容器的 9999 端口，这样就可以通过访问物理主机的 8888 端口来访问容器的 9999 端口了
